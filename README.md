# Basic Network Sniffer (Python)

This project is a simple network packet sniffer written in Python using Scapy.

It captures live network packets and displays them in a table format similar to Wireshark.

Features

- Live packet capture
- Wireshark style packet table
- Color coded protocols
- TCP, UDP and ICMP detection
- Port detection
- DNS query capture
- HTTP request detection


Installation

Install dependencies:

pip install -r requirements.txt

Usage

Run the program:

python network_sniffer.py
